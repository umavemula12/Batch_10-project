Dataset

The proposed project focuses on the automated detection and classification of intracranial hemorrhage (ICH) from non-contrast brain CT images using deep learning techniques. Brain CT scans are widely used in emergency settings due to their speed, availability, and effectiveness in identifying acute hemorrhagic conditions.
The dataset used in this project consists of non-contrast cranial CT images acquired from publicly available medical imaging repositories. Each CT scan contains multiple axial slices of size 512×512 pixels, capturing detailed structural information of the brain. The images are annotated at the slice level to indicate the presence or absence of intracranial hemorrhage, along with hemorrhage subtypes such as subarachnoid, intraventricular, subdural, epidural, and intraparenchymal hemorrhage.
To ensure reliable and unbiased performance evaluation, the dataset is systematically divided into training, validation, and testing subsets. The training set is used to learn discriminative features, the validation set supports hyperparameter tuning and model optimization, and the test set is reserved for final performance assessment. This structured data preparation enables robust model generalization and supports accurate clinical decision assistance.
