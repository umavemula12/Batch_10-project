Pre-processing

Method I: Multi-Window CT Image Enhancement



In the first preprocessing approach, each non-contrast brain CT slice is processed using clinically relevant Hounsfield Unit (HU) windowing techniques to enhance different anatomical structures. Three standard CT windows are applied: brain window (window level = 40, window width = 80), subdural window (window level = 80, window width = 200), and bone window (window level = 600, window width = 2000). These windows emphasize soft tissue, hemorrhagic regions, and skull boundaries respectively.

The three windowed images are concatenated to form a three-channel input representation, enabling the deep learning model to learn complementary features from multiple intensity ranges. To maintain spatial consistency across scans, resampling is applied to standardize pixel spacing. Finally, the images are cropped to retain only the brain region, eliminating redundant background information and improving computational efficiency.




Method II: Adjacent Slice-Based Spatial Encoding



The second preprocessing method captures inter-slice spatial context by utilizing adjacent CT slices. For each target slice, three consecutive slices are selected and combined to construct a pseudo-RGB image. The previous slice (St−1) is mapped to the red channel, the current slice (St) to the green channel, and the next slice (St+1) to the blue channel.

This approach enables the model to learn volumetric continuity and subtle hemorrhagic patterns that may not be visible in a single slice. All slices are processed using the brain window configuration to highlight intracranial structures. Similar to Method I, region-of-interest cropping is applied to focus on diagnostically significant areas, ensuring robust feature extraction and improved classification performance.
